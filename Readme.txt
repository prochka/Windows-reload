Этот Gist описывает что и как нужно ставить после переустановки Windows

1. Sublime Text3
	- устанавливаем нужные плагины.
	- при установке плагина Gutter Color необходимо установить вот этот скрипт http://www.imagemagick.org/script/download.php
	- после установки в настройках Gutter Color USER прописать путь 
	{
		"convert_path" : "/usr/local/bin/convert"
	}
